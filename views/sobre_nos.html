O projeto:

Estamos desenvolvendo um projeto voluntário, um site dedicado às causas animais, para facilitar o acesso a informações sobre eventos de apoio à saúde e bem-estar
dos animais, como adoção, vacinação e castração, oferecidos de forma gratuita ou a baixo custo. Nosso objetivo é apoiar tanto as famílias quanto os animais que
precisam, promovendo iniciativas que fortalecem a comunidade e facilitam o acesso a recursos essenciais para o cuidado animal.
Esse projeto está sendo desenvolvido como parte de um challenge de programação da Twist, envolvendo HTML, CSS e JavaScript.
Para atender aos requisitos, estamos usando tags HTML semânticas para garantir uma estrutura bem organizada e acessível, além de folhas de estilo CSS externas e
JavaScript em arquivos separados para melhor modularidade. Também estamos integrando JSON e LocalStorage, para que as informações sobre eventos e dados do
usuário sejam armazenadas de forma eficiente e acessível. A acessibilidade está no centro do projeto, garantindo que todas as pessoas, independentemente de suas
necessidades, possam usufruir plenamente dos recursos da plataforma.
